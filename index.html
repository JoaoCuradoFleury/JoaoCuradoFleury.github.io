Programa de análise estrutural pelo método dos elementos finitos

Programa de análise estrutural desenvolvido na linguagem python. Esse programa importa os dados de malha e o vetor de carregamentos fornecido pelo usuário, monta a matriz de rigidez e calcula os deslocamentos. Por enquanto essa ferramenta é capaz de calcular e plotar as tensoes para estruturas 2D. A idéia no futuro é ampliar o programa para estruturas 3D, desenvolver uma interface mais amigável ao usuário e desenvolver minhas próprias ferrementas de verificacão de criterios de falha e de verificação normativa, por exemplo.
